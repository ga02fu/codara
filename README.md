# Codara

面向 AI Coding 的本地 Agent Harness，集成请求重写、分层记忆、提示词压缩、工具调用治理与会话持久化能力，支持 Agent 在工作区内完成检索、阅读、修改和执行等操作。

## 核心技术

- Python
- Harness
- 上下文管理
- 结构化记忆
- 工具调用治理

## 核心能力与结果

### 1. 长上下文瘦身

设计上下文管理与裁剪机制，在 16 组长上下文测试中：

- 平均 Prompt 长度：`3435 → 3071`
- 平均压缩率：`11%`

### 2. 多任务请求优化

将请求重写为结构化子任务，并设计约束型提示词，使 Agent 能跳过失败子任务继续执行后续步骤。在 8 组部分失败测试中：

- 任务完成率：`0% → 75%`

### 3. 结构化记忆系统

针对多轮任务中 Agent 重复调用工具的问题，将记忆拆分为：

- **General Memory**：管理长期会话轨迹
- **Detail Memory**：管理工具执行结果、文件摘要与最近文件列表

在“无记忆 / 仅 General / General + Detail”三种设置下，20 个测试任务的平均工具步数分别为：

- `2.8 / 2.0 / 0.4`

### 4. 工具安全与运行治理

构建标准化工具调用与安全边界，覆盖：

- 参数校验
- 工作区隔离
- 高风险审批
- 重复调用拦截

在测试任务中保持：

- Verifier 通过率 `100%`

### 5. 评测与审计闭环

针对上下文治理、多任务请求、记忆系统与工具安全模块设计对比实验任务，搭建效果评测与执行审计闭环，量化验证各模块优化收益。

## 项目结构（示意）
